Trabajamos desde un enfoque integral, fortaleciendo habilidades emocionales, sociales y cognitivas mediante estrategias adaptadas a las necesidades de cada persona.

Servicios Psicológicos en Lima y Callao

Brindamos consulta psicológica, evaluación psicológica y terapia psicológica para niños, adolescentes en Lima y Callao.

① Consulta Psicológica

La consulta psicológica permite identificar de forma clara el motivo de consulta y brindar orientación psicológica profesional.


Es el primer paso para determinar si se requiere una evaluación psicológica o iniciar un proceso de terapia psicológica.

② Evaluación Psicológica Integral para un Diagnóstico Preciso

La evaluación psicológica permite conocer de manera objetiva el estado emocional, conductual, social y cognitivo del paciente mediante pruebas psicológicas estandarizadas y entrevistas clínicas.


Este proceso facilita la elaboración de un diagnóstico preciso y la planificación de una intervención psicológica personalizada.

③ Terapia Psicológica para Niños, Adolescentes

La terapia psicológica es un tratamiento profesional orientado a trabajar las dificultades identificadas durante la evaluación psicológica.


Se intervienen problemáticas como ansiedad, depresión, dificultades emocionales, conductuales, familiares, escolares y relacionales, promoviendo el bienestar y la salud mental.
